# Overview
Browser-based implementation Minesweeper. You win by avoiding all the hidden mines and revealing all the safe tiles. Left click reveals a tile. Numbers on a revealed tile indicate how many mines are directly around it. Right click a tile to place a flag on suspected mines. Doing so will ensure that corresponding mines cannot be detonated. Clicking a mine will end the game.

# Features
Three difficulty levels: Easy, Medium, Hard. A first click guard to ensure you can't lose on the first tile reveal. A game statistics panel that shows how many mines there are and remaining safe tiles.

# Installation
Download the project files. Ensure the following files are in the same folder: minesweeper.html, minesweeper.css, minesweeper.js, bomb.png and flag.png. Then open minesweeper.html in a web browser.
